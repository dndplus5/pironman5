Batocera Linuxのインストール
======================================================

|link_batocera| はオープンソースで完全無料のレトロゲーミングディストリビューションで、USBスティックやSDカードにコピーして、任意のコンピュータやナノコンピュータを一時的にまたは永久にゲームコンソールに変えることができます。

Micro SDまたはNVMe SSDのどちらを持っているかに基づいて、インストール方法を選択できます。

NVMe SSDへの直接インストールには、Micro SDカードへのインストールに比べて追加の手順が必要です。それは、Raspberry Piのブートローダーを更新し、NVMe SSDからのブートを優先するようにすることです。デフォルトでは、Micro SDカードからのブートが優先されています。
